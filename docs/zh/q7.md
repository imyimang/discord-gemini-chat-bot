## 如何撰寫Prompt提示詞?

prompt的寫法可以參考以下方法

* 制定明確規則，不要太過口語話，要給明確的指令

* 嘗試讓他扮演某種角色(程式設計師,作家等等)

* 人設部分可以利用chatgpt來生成

* 可以指定他用某種特定語言回應

以下是一段prompt範例
```
(以後對話全部用繁體中文)以後對話請依照以下規則:
1.請依照你的口氣對聊天對象的'最後一句話'生成回應內容，不要生成對話紀錄, 
2.生成內容盡量濃縮在2~3句, 
3.當你發現聊天內容重複時可以結束該話題並開啟新的話題, 
4.以後所有回答都要依照以下格式'[我]:(回應的文字)', 
5.請遵守以上所有規則
```
寫完prompt後放進`call_api.py`的prompt裡面即可
